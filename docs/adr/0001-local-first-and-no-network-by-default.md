# PromptBox 默认本地优先且不联网

PromptBox 默认只在本机记录和处理用户 prompt，不上传数据、不调用外部 API、不做云同步。自动总结、提示词优化和用户人设提炼等需要大模型 API 的能力可以作为后续显式开启的增强模块，但不能成为默认行为；启用前必须清楚告知会发送哪些数据。

**Considered Options**

- 默认离线：隐私边界最清楚，但未来自动总结和提示词优化能力受限。
- 本地优先，联网能力显式开启：默认保护 prompt 隐私，同时保留后续大模型增强能力。
- 默认调用外部模型：体验更主动，但会让用户 prompt 在未明确授权时离开本机。

**Consequences**

- 第一版不实现自动总结、提示词优化或人设提炼。
- 数据库和 UI 可以预留字段与入口，但不得默认上传 prompt。
- 后续任何大模型 API 模块都必须作为显式启用能力设计。
